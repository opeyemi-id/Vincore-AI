# VAI-BRD-001 Brand Guidelines
**Version:** 1.0 | **Owner:** Opeyemi Idonuagbe | **Status:** Active

---

## Logo

**The mark:** A geometric V shape — two arms converging at a single teal core point. The bottom dot represents the "core" in VinCORE. One top node is filled (input), one is outlined (output) — shorthand for what Vincore does.

**The wordmark:** "VINCORE" in deep navy + "AI" in electric teal, same weight.

**Logo rules:**
- Minimum size: 20px height for icon mark, 32px for full lockup
- On dark backgrounds: use white arms + teal dot
- Never stretch, rotate, recolor, or add effects
- Never place on busy photographic backgrounds without an overlay

---

## Colour Palette

| Name | Hex | Usage |
|---|---|---|
| Deep Navy | #0F1E3C | Primary — headings, nav, dark sections, buttons |
| Navy Mid | #1B3161 | Hover states on navy elements |
| Electric Teal | #00C9B1 | Accent — CTAs, highlights, logo AI mark |
| Off-White | #F8F9FA | Page background, card backgrounds |
| White | #FFFFFF | Card surfaces |
| Text | #1A1A2E | Body text |
| Muted | #6B7280 | Subtext, descriptions |

**Never use:** Pure black backgrounds, purple, lime green, generic startup blue.

---

## Typography

**Font:** Inter (Google Fonts) — weights 300, 400, 500, 600, 700, 800, 900

**Type scale:**
- Hero headline: clamp(38px, 7vw, 68px) / weight 900
- Section title: clamp(28px, 5vw, 44px) / weight 900
- Eyebrow: 11px / weight 700 / tracking 3px / uppercase / teal
- Body: 16–17px / weight 400 / line-height 1.75

---

## Brand Personality

**Three words:** Precise. Capable. Founder-led.

Vincore is not a flashy tech startup. It is a serious, ISO-trained operator who uses AI and automation. Premium without being cold. Confident without being arrogant.
