# Brand guidelines

## Logo

**The mark:** A geometric V shape — two arms converging at a single teal core point. The bottom dot represents the "core" in VinCORE: the central intelligence where all systems converge. One top node is filled (input), one is outlined (output) — shorthand for what Vincore does.

**The wordmark:** "VINCORE" in deep navy + "AI" in electric teal, same weight. Reads as one name, carries two meanings.

**Logo files:** Stored in `docs/brand/` — use the HTML/SVG source in `website/index.html` nav as reference.

---

## Color palette

| Name | Hex | Usage |
|---|---|---|
| Deep Navy | `#0F1E3C` | Primary — headings, nav, dark sections, buttons |
| Navy Mid | `#1B3161` | Hover states on navy elements |
| Electric Teal | `#00C9B1` | Accent — CTAs, highlights, logo AI mark, links |
| Teal Dim | `rgba(0,201,177,0.11)` | Teal backgrounds (tags, icon backgrounds) |
| Teal Glow | `rgba(0,201,177,0.28)` | Button box shadows |
| Off-White | `#F8F9FA` | Page background, card backgrounds |
| White | `#FFFFFF` | Card surfaces, contact section |
| Text | `#1A1A2E` | Body text |
| Muted | `#6B7280` | Subtext, descriptions, labels |
| Border | `rgba(15,30,60,0.09)` | Card borders, dividers |

**Never use:** Pure black backgrounds, purple, lime green, generic startup blue.

---

## Typography

**Font:** Inter (Google Fonts) — weights 300, 400, 500, 600, 700, 800, 900

**Type scale:**
- Hero headline: `clamp(38px, 7vw, 68px)` / weight 900 / tracking -1.8px
- Section title: `clamp(28px, 5vw, 44px)` / weight 900 / tracking -0.6px
- Eyebrow: `11px` / weight 700 / tracking 3px / uppercase / teal
- Body: `16–17px` / weight 400 / line-height 1.75
- Small/labels: `11–13px` / weight 600–700

---

## Logo usage rules

- Minimum size: 20px height for icon mark, 32px for full lockup
- Clear space: always leave space equal to the height of the "V" arm around the logo
- On dark backgrounds: use white arms + teal dot (see dark variant in logo file)
- Never stretch, rotate, recolor, or add effects to the logo
- Never place on busy photographic backgrounds without an overlay

---

## Brand personality

**Three words:** Precise. Capable. Founder-led.

Vincore is not a flashy tech startup. It is a serious, ISO-trained operator who happens to use AI and automation — and the brand should feel that way. Premium without being cold. Confident without being arrogant. Human without being informal.
