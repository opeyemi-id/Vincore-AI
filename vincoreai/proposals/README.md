# Proposals

This folder contains client proposals, one file per prospect.

## Naming convention

`YYYY-MM-DD-client-name.md`

Example: `2026-07-15-swiftfix-hvac.md`

## Status tags

Add one of these at the top of each proposal file:
- `[DRAFT]` — being written
- `[SENT]` — sent to client, awaiting response
- `[ACCEPTED]` — client accepted, project starting
- `[DECLINED]` — client declined
- `[EXPIRED]` — no response after 14 days

*(Proposals will be added here as client conversations progress)*
