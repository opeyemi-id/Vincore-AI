# 11 AI Agents

The Vincore Intelligence Platform. This folder contains everything needed to build, govern, and operate AI agents for Vincore AI.

---

## Structure

```
11-ai-agents/
├── 01-agent-governance/       ← Charter, permissions, boundaries, compliance rules
├── 02-vincore-knowledge-base/ ← Everything agents need to know about Vincore
├── 03-agent-personas/         ← Individual agent profiles and system prompts
├── 04-prompt-library/         ← Organised by function (sales, marketing, ops)
├── 05-visual-assets/          ← Headshots, brand images, avatar variations
├── 06-videos/                 ← LinkedIn, Instagram, demos, case studies
├── 07-voice/                  ← Voice samples, ElevenLabs files, scripts
├── 08-conversations/          ← Example conversations for training data
├── 09-performance/            ← Monthly KPIs, quarterly reports, experiments
├── 10-improvements/           ← Version history, prompt improvements, A/B tests
└── 11-archive/                ← Retired prompts, old avatars, old workflows
```
